🌿 AfriFit

Healthy Living, African Style

AfriFit is a mobile health and nutrition app designed for Africa. It helps users track calories, water intake, weight, exercise, and healthy habits using foods commonly eaten across African countries.

✨ Features

- 🔥 Daily calorie calculator
- 🍽️ Meal tracking
- 💧 Water intake tracker
- ⚖️ Weight tracker
- 📈 Progress charts
- 🥗 African food database
- 🤖 AI-powered nutrition features (coming soon)

🛠️ Built With

- Flutter
- Firebase
- Cloud Firestore
